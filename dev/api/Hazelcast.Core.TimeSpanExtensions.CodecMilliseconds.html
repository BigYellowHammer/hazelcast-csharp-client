    The time span.
    Int64 infinite

    The value of "infinite".
    Returns
    Int64

    The value of the specified time span expressed in whole milliseconds.
    Remarks

    For timeouts, codecs expect infinite to be zero, but for lease times, or other durations, they expect infinite to be -1s or sometimes MaxValue. This method converts Zero to zero, and InfiniteTimeSpan to infinite.

    This allows our public API to use InfiniteTimeSpan (which has an actual value of -1ms) for everything infinite without bothering about other conventions.
