Remarks
Puts an entry into this map. Similar to Put(TKey, TValue) operation except that Set doesn't return the old value which is more efficient.

Warning:

This method uses GetHashCode and Equals of binary form of the key, not the actual implementations of GetHashCode and Equals defined in key's class.
